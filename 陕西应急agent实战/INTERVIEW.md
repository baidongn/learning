# 求职演示与面试索引

## 5 分钟演示脚本

1. 30 秒：说明问题——高速事件信息不完整、工具多、动作风险高。
2. 60 秒：展示 12 周渐进路线，强调 Supervisor 最后开发。
3. 90 秒：在 Vue 指挥台提交秦岭隧道烟雾案例，展示四专业 Agent 轨迹。
4. 60 秒：解释 RAG 引用、模拟 Tool、MCP、Checkpoint 与人工审批。
5. 40 秒：展示安全 BLOCK、`executed_actions=[]` 和 Prompt 注入测试。
6. 20 秒：展示 20 场景评测与 Docker/Kustomize 交付。

## 简历描述示例

> 独立设计陕西高速路网应急指挥 Agent 教学项目，基于 FastAPI、LangGraph、PostgreSQL/pgvector、MCP 与 DeepSeek 构建 4 个专业 Agent + 1 个受限 Supervisor；实现 RAG 引用、模拟 Tool、Checkpoint/HITL、安全复核、20 场景真实编排评测、Vue 指挥台及 Docker/Kustomize 部署，Mock 基线四项指标为 100% / 100% / 100% / 0%。

## 必讲取舍

- 为什么先固定工作流，再做 Supervisor。
- 为什么预案引用由检索代码绑定，而不接受模型生成引用。
- 为什么资源调度只生成建议、不编造资源、不执行真实动作。
- 为什么 PostgreSQL 是主库，Redis 只做缓存/临时状态。
- 为什么安全红线单独验收，不能被平均分掩盖。

每周课程末尾另有三道对应面试题。
